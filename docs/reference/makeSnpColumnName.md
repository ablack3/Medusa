# Build the canonical cohort-data column name for an instrument SNP

<div class="ref-description section level2">

Build the canonical cohort-data column name for an instrument SNP

</div>

<div class="section level2">

## Usage

<div class="sourceCode">

``` r
makeSnpColumnName(snpId)
```

</div>

</div>

<div class="section level2">

## Arguments
