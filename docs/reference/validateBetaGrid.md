# Validate Beta Grid

<div class="ref-description section level2">

Checks that a beta grid is a sorted numeric vector with sufficient
resolution.

</div>

<div class="section level2">

## Usage

<div class="sourceCode">

``` r
validateBetaGrid(betaGrid)
```

</div>

</div>

<div class="section level2">

## Arguments

-   betaGrid:

    Numeric vector of grid points.

</div>

<div class="section level2">

## Value

Invisible TRUE if valid; throws an error otherwise.

</div>
