# Validate Instrument Table Structure

<div class="ref-description section level2">

Checks that an instrument table has the required columns and valid data
types.

</div>

<div class="section level2">

## Usage

<div class="sourceCode">

``` r
validateInstrumentTable(instrumentTable)
```

</div>

</div>

<div class="section level2">

## Arguments

-   instrumentTable:

    A data frame to validate.

</div>

<div class="section level2">

## Value

Invisible TRUE if valid; throws an error otherwise.

</div>
