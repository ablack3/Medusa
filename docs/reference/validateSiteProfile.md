# Validate Site Profile Object

<div class="ref-description section level2">

Checks that a site profile object has the required structure.

</div>

<div class="section level2">

## Usage

<div class="sourceCode">

``` r
validateSiteProfile(siteProfile)
```

</div>

</div>

<div class="section level2">

## Arguments

-   siteProfile:

    A list to validate.

</div>

<div class="section level2">

## Value

Invisible TRUE if valid; throws an error otherwise.

</div>
